## 贡献

如果你真的想要发一个 Pull Request，这里是一些你需要的信息：

- `npm run dev` 用开发模式启动 Epherome
- `npm run prettier` 自动美化您的代码
- `npm run eslint` 检查您的代码
- `npm test` 测试
